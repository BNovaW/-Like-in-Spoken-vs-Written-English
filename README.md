# -Like-in-Spoken-vs-Written-English

Author: Brooke Watson

Date: Dec 2025

**Description:**

This is a corpus linguistics class project examining the usage of "like" in spoken and written English. 
	This project combines sociolinguistics and pragmatics to look at the usage of the word “like” in a spoken versus written context. Pragmatics and more specifically comparing written and spoken contexts is commonly studied. These types of studies often focus on syntax, structure, grammar, and more abstractly communicative function. Sociolinguistic studies often focus on uncommon words or phrases and try to get at what influences the structures and usage choices of language. By combining these two areas of research I will focus on the context “like” occurs in, the frequency of usage, grammatical roles, and usage choices to attempt to conclude if “like” is used in a different way in spoken versus written English. 

Data Sources: 

- Santa Barbara Corpus of Spoken American English parts 1-4 found here: https://www.linguistics.ucsb.edu/research/santa-barbara-corpus-spoken-american-english
- MASC corpus found here: https://anc.org/data/masc/corpus/

**Methods**
This project uses three main linguistic analysis methods, frequency analysis, collocation, and keyness. These are described in more detail in the "Final Project Paper BW" PDF. 

**How To?**
The code for this project uses python and requires the libraries re, glob, math, operator, os, random, and collections. To run the python file you will need the data files as well as working python and the libraries listed before.
